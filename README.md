  Meadows-CPP
</h1>

<!-- description -->
<p align="center">
  <strong> A C++ package for simulating electric field driven spark discharges in air with finite volume methods on cartesian grids</strong>
</p>

Meadows is a finite volume based simulation package written in C++ to be able to resolve multiphysics, inspiration taken from other excellent pieces of software such as: 
- [AFIVO](https://github.com/MD-CWI/afivo-streamer)
- [Chombo Discharge](https://github.com/chombo-discharge/chombo-discharge)
- [Trixi.jl](https://github.com/trixi-framework/Trixi.jl/tree/main)
- [Oceananigans.jl](https://github.com/CliMA/Oceananigans.jl/tree/main)
